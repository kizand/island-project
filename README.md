com.javarush.island

Пакеты и классы:
1) organisms - пакет с классами животных, растений и насекомого
- Organism
- OrganismCreator
- Predator (хищник)
- Herbivore (травоядное)
- Insect (насекомое)
- Plant (растение)
2) island - пакет с классами остров и клетка
- Island
- Cell (клетка)
3) config - конфигурация приложения
- ApplicationConfig
- ConsoleConfig
- GlobalOrganismConfig
- IslandConfig
- OrganismConfig
- Settings
4) component - компоненты приложения
- Application
- GlobalOrganismList
- OrganismGroup
- OrganismGroupsIterator
- Children
- Utils
5) view - пакет с классами по виду приложения
- View
- ConsoleView
6) thread - пакет с классами по многопоточности
- ThreadAction
- ThreadWorker
7) resources - пакет с файлом yaml с настройками
- settings.yaml

StartConsole - главный класс для запуска (точка входа)
